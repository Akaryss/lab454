# Лабораторная работа №4(5) - ASP.NET Core MVC (Identity & Testing)

![Build and Test Status](https://github.com/Akaryss/Lab454/actions/workflows/dotnet-build.yml/badge.svg)

## Описание проекта
Данное приложение представляет собой сервис объявлений, разработанный на платформе ASP.NET Core MVC с использованием Entity Framework Core.

## Выполненные требования:
1. **Аутентификация и авторизация**: 
   - Интегрирован ASP.NET Core Identity.
   - Реализованы две ролевые политики: `Admin` и `User`.
2. **Управление состоянием и производительность**:
   - Настроен именованный профиль кэширования в `Program.cs` (время жизни кэша 290 сек для варианта 25).
   - К контроллерам применен атрибут `[ResponseCache]`.
3. **Администрирование**:
   - Реализован `AdminController` для управления пользователями (просмотр, редактирование, удаление).
   - Реализована модерация объявлений и категорий.
4. **Навигация и UI**:
   - Использование мастер-страниц (`_Layout.cshtml`) для единого стиля.
   - Постраничный вывод данных (пагинация) и фильтрация по полям на странице объявлений.
5. **Тестирование**:
   - Создан проект модульных тестов на базе **xUnit** и **Moq**.
   - Покрыты тестами методы контроллеров `Advertisements`, `Categories`, `Users`, `Home`.
   - Результат: **7 тестов пройдено успешно**.
6. **CI/CD**:
   - Настроен рабочий процесс **GitHub Actions** для автоматической сборки и тестирования на платформах Windows и Linux.
